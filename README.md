# My Todo List App

이 프로젝트는 간단한 투두 리스트 애플리케이션을 React로 구현한 것입니다.  
기능으로는 할 일 추가, 삭제와 체크박스 체크를 통해 줄긋기가 있습니다.

## 기능

- 할 일 추가
- 할 일 삭제
- 할 일 완료 상태 표시(줄긋기)

## 기술 스택

- React
- JavaScript
- CSS
- HTML
